# GreenTBM v0.6 검색 안정화 버전

## 수정 내용
- 검색 시 먼저 내장된 서울 주요 공원/녹지/거점 좌표를 확인합니다.
- 내장 장소에 없으면 네이버 Geocoding으로 주소를 검색합니다.
- 네이버 Geocoding이 장소명 검색에 실패해도 서울숲, 보라매공원, 올림픽공원 등은 바로 이동합니다.
- 검색창 아래 추천 장소 버튼을 추가했습니다.
- 지도 상태 영역에 지오코딩 모듈 로드 상태와 검색 방식을 표시합니다.
- Smartway CCTV 링크와 기상청 날씨/기상특보 링크는 유지했습니다.

## 네이버 클라우드 확인
- Web Dynamic Map 활성화
- Geocoding 활성화
- Web 서비스 URL에 Vercel 주소 등록
- 배포 후 Vercel에서 최신 배포가 성공했는지 확인
