# GreenTBM v0.2

녹지관리, CCTV, 공종별 TBM 체크리스트 통합 웹앱입니다.

## 이번 버전 반영사항

- `공종별 TBM 체크리스트.xlsx` 원본 데이터 반영
- 공통사항은 항상 표시
- 공종 선택 시 해당 공종의 `TBM 체크리스트` 원문 표시
- 미사여구 추가 없이 엑셀 파일의 원본 문구 그대로 표시
- 네이버지도 기본 연동 구조 포함

## 네이버 지도 키

기본값은 코드에 들어가 있습니다. Vercel에서 환경변수로 바꾸려면 아래 이름을 사용하세요.

```txt
VITE_NAVER_MAP_KEY
```
