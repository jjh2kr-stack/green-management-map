# GreenTBM v1.0 관할판별 최종본

네이버지도 검색 위치가 업로드 PDF 기준 자동차전용도로 녹지관리 관할 구역인지 판별하는 버전입니다.

## 사용 방법
1. `src/config.js`에서 `NAVER_CLIENT_ID` 값을 본인 네이버 클라우드 Client ID로 바꾸세요.
2. GitHub 저장소의 기존 파일을 삭제하고, 이 ZIP 압축 해제 파일 전체를 업로드하세요.
3. Vercel에서 자동 배포되면 사용하세요.

## 판별 방식
- 내장 장소/교량/IC 검색
- 네이버 Geocoding 주소 검색
- 검색 좌표와 관할 노선 중심선 거리 비교
- 관할 반경 기본값: 250m
