# GreenTBM v0.4

변경사항

- 작업구역 검색 기능 개선
  - 네이버 Geocoding 결과를 우선 사용
  - 공원/녹지명 검색이 실패할 경우 내장된 서울 주요 녹지 좌표로 이동
- 검색 위치 기준 반경 500m CCTV 목록 갱신
- CCTV 확인 링크를 Smartway로 변경
  - https://smartway.seoul.go.kr/web/main.view
- 오른쪽 하단 날씨 카드에 기상청 날씨와 기상특보 확인 버튼 추가
- 공통사항 + 공종별 체크리스트 원본 문구 유지

주의

- 임의 장소명까지 모두 검색하려면 네이버 클라우드 Application에서 Geocoding API도 활성화해야 합니다.
- Smartway CCTV는 현재 새 창으로 열어 확인하는 방식입니다. 외부 사이트 정책상 iframe 내 직접 재생은 제한될 수 있습니다.
- 기상특보 원문 자동 표시는 기상청 API 허브 또는 공공데이터포털 API 키가 필요합니다. 현재 버전은 기상청 특보 화면으로 바로 연결합니다.
