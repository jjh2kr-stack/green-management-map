# GreenTBM v0.3

녹지관리, 작업구역 검색, 서울도시공간포털 CCTV 확인, 기상청 날씨 링크, 공종별 TBM 체크리스트 통합 웹앱입니다.

## 변경 내용

- 작업구역 검색 추가
- 검색 결과 좌표 기준 지도 이동
- 검색 결과 기준 주변 CCTV 목록 갱신
- CCTV 마커/목록 클릭 시 서울도시공간포털 열기
- 오른쪽 하단 날씨 카드 추가
- 기상청 날씨 클릭 연결
- 공통사항은 항상 표시
- 공종별 체크리스트는 원본 문구 그대로 표시

## 배포

기존 GitHub 저장소의 파일을 이 버전 파일로 교체 후 Vercel에서 자동 배포됩니다.

## 네이버 지도 키

Vercel 환경변수에 `VITE_NAVER_MAP_KEY`를 등록하거나, 코드의 기본값을 사용합니다.
네이버 클라우드 Maps Web 서비스 URL에 Vercel 배포 주소를 추가해야 합니다.
